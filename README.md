# Kodbud-AI-Internship
# AI Internship Task 1: Spam Email Classifier

## Description
This project was developed as part of the **Kodbud Tech-Driven Internship Program**. The goal is to build a machine learning model that classifies SMS/Email messages as 'Spam' or 'Ham' (Normal) using Natural Language Processing (NLP) techniques.

## Features
* **Exploratory Data Analysis (EDA):** Visualizing class distribution and data balance.
* **TF-IDF Vectorization:** Converting text data into numerical features based on importance.
* **Naive Bayes Classifier:** Utilizing the Multinomial Naive Bayes algorithm for high-accuracy classification.
* **Predictive System:** A function to test custom messages.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-learn, Matplotlib, Seaborn
* **Environment:** Google Colab 

## Results
* **Accuracy:** 96.68%
* **Model:** Multinomial Naive Bayes
